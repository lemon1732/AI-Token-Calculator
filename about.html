AI Token Cost Calculator

About

AI Token Cost Calculator is a free, open-source tool designed to help developers, data scientists, and teams make informed decisions about AI API costs.

What it does

  • Cost Calculator — Compare pricing across 20+ models from OpenAI, Anthropic, Google, and DeepSeek. Calculate input, output, cache write, and cache read costs with a single click.
  • Token Counter — Paste any text and get instant token counts powered by tiktoken, the same tokenizer used by OpenAI's API. Non-OpenAI models show reference estimates.

Data accuracy

All pricing data is sourced from official vendor websites and updated monthly. The token counter uses js-tiktoken, a pure JavaScript implementation of OpenAI's tokenizer, for accurate token counting on GPT models.

Privacy

All calculations run entirely in your browser. No text, tokens, or usage data are sent to any server. See our Privacy Policy for details.
