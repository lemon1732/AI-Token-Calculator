AI Token Cost Calculator

About

AI Token Cost Calculator is a free tool that helps developers, data scientists, and teams compare AI API pricing and estimate costs before writing a single line of code. We track pricing across OpenAI, Anthropic, Google, and DeepSeek — 20+ models, updated regularly.

What we offer

  • Cost Calculator — Compare pricing across all major AI providers. Calculate input, output, and cache read costs instantly.
  • Token Counter — Paste any text and get instant token counts. Uses js-tiktoken for accurate GPT tokenization.
  • Blog — In-depth guides on AI API pricing, prompt caching, model selection, and cost optimization strategies.

Data accuracy

All pricing data is sourced directly from official provider documentation. We verify prices against each provider's pricing page and update when changes are announced. Each article includes a "last verified" note with the source date.

Found a pricing error? Please report it — we correct mistakes within 24 hours.

How we work

We are an independent project with no affiliation to any AI provider. We do not receive referral fees or commissions. Our goal is to provide accurate, unbiased information to help you make better decisions about AI API costs.

Privacy

All calculations run entirely in your browser. No text, tokens, or usage data are sent to any server. See our Privacy Policy for details.
